# Database

Databases and data storage solutions in the CNCF landscape.

- [MongoDB](mongodb.md) — Distributed document database for modern applications
- [MySQL](mysql.md) — The world's most popular open-source relational database
- [PostgreSQL](postgresql.md) — Advanced open-source relational database with extensibility
- [Redis](redis.md) — In-memory data store for caching, messaging, and real-time analytics
- [TiKV](tikv.md) — Distributed transactional key-value database built on RocksDB
