# Streaming & Messaging

Message brokers and event streaming platforms for asynchronous communication and data pipelines.

- [EMQX](emqx.md) — Scalable MQTT broker for IoT and real-time messaging
- [Kafka](kafka.md) — Distributed event streaming platform for high-throughput data pipelines
- [RabbitMQ](rabbitmq.md) — Feature-rich open-source message broker supporting multiple protocols
- [RocketMQ](rocketmq.md) — Distributed messaging and streaming platform with low latency
