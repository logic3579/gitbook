# Coordination & Service Discovery

Distributed coordination, configuration management, and service discovery systems.

- [CoreDNS](coredns.md) — Flexible and extensible DNS server for service discovery in Kubernetes
- [Etcd](etcd.md) — Distributed key-value store for shared configuration and service discovery
- [Nacos](nacos.md) — Dynamic service discovery, configuration, and management platform
- [ZooKeeper](zookeeper.md) — Centralized service for distributed synchronization and configuration
