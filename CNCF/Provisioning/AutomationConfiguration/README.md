# Automation & Configuration

Infrastructure automation, configuration management, and provisioning tools.

- [Ansible](ansible.md) — Agentless automation platform for configuration management and orchestration
- [Apollo](apollo.md) — Distributed configuration management center for microservices
- [OpenStack](openstack.md) — Open-source cloud computing platform for building private and public clouds
- [Salt Project](saltproject.md) — Event-driven automation and configuration management framework
- [Terraform](terraform.md) — Infrastructure as Code tool for provisioning cloud resources declaratively
