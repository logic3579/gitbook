# Cloud Native Storage

Storage solutions for persistent data in containerized and Kubernetes environments.

- [CSI](csi.md) — Container Storage Interface specification for exposing storage to containerized workloads
- [MinIO](minio.md) — High-performance S3-compatible object storage for cloud-native workloads
- [Rook](rook.md) — Storage orchestrator for Kubernetes, turning distributed storage into self-managing services
- [Velero](velero.md) — Backup and disaster recovery tool for Kubernetes cluster resources and volumes
