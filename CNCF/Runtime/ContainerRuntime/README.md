# Container Runtime

Container runtimes for building, running, and managing containers and container images.

- [Docker](docker.md) — Industry-standard platform for building and running containerized applications
- [Containerd](containerd.md) — Industry-standard container runtime focused on simplicity and portability
- [CRI-O](cri-o.md) — Lightweight container runtime specifically designed for Kubernetes
- [Lima](lima.md) — Linux virtual machine manager for running containers on macOS
