# Cloud Native Knowledge Base

A technical knowledge base covering cloud-native technologies, DevOps practices, and platform engineering. Documentation is organized following the [CNCF landscape](https://landscape.cncf.io/) taxonomy.

## Sections

- [**CNCF**](./CNCF/README.md) — Cloud-native tools organized by the CNCF landscape taxonomy: CI/CD, databases, messaging, observability, service mesh, container runtime, and more
- [**DevOps**](./DevOps/README.md) — Programming languages, CLI references, network fundamentals, service configurations, and Linux system topics
- [**Platform**](./Platform/README.md) — Cloud provider guides (AWS, GCP, Aliyun) and Kubernetes distribution installers
- [**Standards**](./Standards/README.md) — Engineering standards: naming conventions, Git Flow, GitHub/GitLab/Docker standards, release automation
- [**Misc**](./Misc/README.md) — VPN/tunnel technologies and interview preparation
- [**Environment**](./Environment/README.md) — Development environment setup
